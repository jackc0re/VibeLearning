# Answers

<details>

1. **B** - Reports transform raw data into insights for decision-making.
2. **B** - Executive summary comes first so readers get key points quickly.
3. **A** - KPI = Key Performance Indicator (important metrics).
4. **B** - CSV is best for further analysis as it's structured and importable.
5. **B** - Context comes from comparing to previous periods or targets.
6. **B** - Exception reports highlight unusual/outlier data needing attention.
7. **B** - Pie charts show proportions; line charts show trends over time.
8. **B** - Headers should include title, date, and data source for context.
9. **B** - Python's `csv` module is the standard for CSV file operations.
10. **A** - Too much data without context overwhelms rather than informs.
