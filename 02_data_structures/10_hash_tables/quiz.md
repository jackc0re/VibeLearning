## Question 1
What does a hash function do?

A) Sorts data  
B) Converts key to integer  
C) Encrypts data  
D) Compresses data

<details>
<summary>Click for answer</summary>

## Question 3
What is a collision?

A) Table is full  
B) Two keys hash to same index  
C) Key not found  
D) Hash function fails

<details>
<summary>Click for answer</summary>

**B) Two keys hash to same index**

Collisions are handled by chaining or probing.

</details>

Python dicts are?

A) Arrays  
B) Linked lists

<details>
<summary>Click for answer</summary>

Python dictionaries use hash tables internally.

## Question 5
Chaining handles collisions by?

A) Finding next slot  
B) Storing list at bucket  
C) Resizing table  
D) Rejecting key

<details>
<summary>Click for answer</summary>

**B) Storing list at bucket**

Each bucket holds a list of all key-value pairs that hash there.

</details>
